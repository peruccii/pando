# Git Specs Index

Este diretorio concentra as especificacoes derivadas de `GIT_PRD.md`.

Arquivos:

- `product_scope.md`: escopo, metas, nao-metas e definicoes.
- `history_engine.md`: historico linear com streaming, cursor e virtualizacao.
- `working_tree_staging.md`: status, stage/unstage, hunk staging e line staging.
- `diff_viewer.md`: diff unified/split, sincronizacao de scroll e limites de arquivo.
- `merge_conflict_management.md`: deteccao e resolucao de conflitos.
- `command_queue.md`: fila sequencial de comandos write por repositorio.
- `filewatcher_integration.md`: refresh orientado a eventos, sem polling.
- `api_contract.md`: contrato de bindings Wails e modelos.
- `ux_shortcuts.md`: UX, atalhos e feedback de acoes.
- `performance_reliability.md`: metas de performance e confiabilidade.
- `observability_security.md`: logs, diagnostico e guardrails de seguranca.
- `test_strategy.md`: estrategia de testes backend/frontend/e2e.
- `roadmap_delivery.md`: fases de entrega e criterios de readiness.
